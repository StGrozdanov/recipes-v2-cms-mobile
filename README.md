# Recipes Mobile

![GitHub last commit](https://img.shields.io/github/last-commit/StGrozdanov/recipes-v2-cms-mobile)

This client is made with React Native, Typescript and Expo.

## Project Overview

### Technologies Used

- :art: Sass 
- ☑️ TypeScript 
- 🧪 Detox & Jest
- 🪝 pre-commit and pre-push hooks
- Expo

## Features

- CI/CD pipeline consisting of typechecking, custom linters, unit tests, integration tests, e2e tests and a single deployment environment
- Responsive design with Sass
- Various frontend libraries for an enhanced user experience
- White and Dark theme
- Websocket integration
- Push Notifications
- Charts
